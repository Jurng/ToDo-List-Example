# Examples

# ToDo List 

```
git clone https://github.com/Jurng/ToDo-List-Example.git

cd ToDo-List-Example/
npm install
npm start

open http://localhost:8100/
```

# What the ToDo-List do ? 

  - All TODO TASK list show on the homepage .
  - User can add the new TODO task ( task title ).
  - User can update their task title.
  - User can mark their task done.
  - User can remove their tasks.
  - Persistent Data 
  - Show number of completed task.
